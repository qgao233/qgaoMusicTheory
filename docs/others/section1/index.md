# 简谱

简谱比五线谱书写简单，方便。

简谱|1|2|3|4|5|6|7
:-:|:-:|:-:|:-:|:-:|:-:|:-:|:-:
唱名|do|re|mi|fa|sol|la|si
音名|C|D|E|F|G|A|B

>**和弦**和**调式**会用到音名
