3. 重音

如果音符上有重音标记，那么在弹奏这个音符时，力度相对来说要更大一点。

人声的话，唱得坚强有力一点。

4. 渐弱、渐强

  • 渐弱：人声越来越弱，弹奏力度越来越小；
  • 渐强：人声越来越强，弹奏力度越来越大。

5. 琶音

琶音( arpeggio)指一串和弦音从低到高或从高到低依次连续奏出，可视为分解和弦的一种。
